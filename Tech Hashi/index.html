what i do

web design (Figma)

I create beautiful, dark-themed interfaces in Figma. From wireframes to high-fidelity prototypes, I focus on user experience, typography, and premium aesthetics with gold accents.

HTML & CSS

Semantic, clean HTML and modern CSS (flex, grid, animations). I hand-code responsive layouts that look perfect on every device. No templates, just custom code.

JavaScript

Interactive experiences with vanilla JS. I add dynamic elements, smooth transitions, and functionality that enhances the user experience without unnecessary bloat.

responsive & premium

Dark blue, black, white and gold — my signature palette. I build websites that feel exclusive, modern, and are fully responsive from mobile to 4K displays.

how i work

1

discovery & figma design

We discuss your vision, brand, and goals. I create wireframes and high-fidelity designs in Figma, incorporating dark themes and gold accents. You see exactly what you'll get before I write a single line of code.

2

html & css development

I translate the design into semantic HTML and modular CSS. I focus on responsive layouts, performance, and cross-browser compatibility. Every detail from Figma is carefully implemented.

3

javascript interactivity

I bring the site to life with JavaScript — smooth animations, interactive elements, and dynamic content. I write clean, efficient code that enhances the user experience.

4

testing & launch

I test on multiple devices and browsers, ensuring everything is pixel-perfect. Then I deploy the site and provide ongoing support. Your project is handled personally from start to finish.
